■使用方法

動画リンク/ID欄にYouTube動画のリンクを入力し、「読み込む」ボタンを押下  
再生ウィンドウに動画が表示されると、再生可能

*各機能紹介*  
再生・停止：動画を再生・停止する  
逆再生：動画を逆再生する  
左右反転：動画を左右反転する  
ループ：動画を繰り返し再生する  
速度：動画の再生速度を変更する  
区間ループ開始：「始まり」欄に入力された秒数から、「終わり」欄に入力された秒数までを繰り返し再生する(分:秒で入力)  
↩：現在の再生位置を始まり、終わりに設定する
区間ループ解除：設定した区間の繰り返しを停止する  
設定を保存：現在の再生設定を「設定名」に入力されたタイトルを付けて保存する  
全て削除：保存された設定を全て削除する  
保存済み設定→読込：保存された設定1件を読み込んで表示する  
保存済み設定→削除：保存された設定1件を削除する
